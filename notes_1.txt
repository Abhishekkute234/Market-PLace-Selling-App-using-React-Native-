wrap it like this 

 <ClerkProvider
      publishableKey={process.env.EXPO_PUBLIC_CLERK_PUBLISHABLE_KEY}
    >
      <SignedIn>
        <Stack>
          <Stack.Screen
            name="(tabs)"
            options={{
              headerShown: false,
            }}
          />
        </Stack>
      </SignedIn>

      <SignedOut>
        <text>Signout</text>
      </SignedOut>
    </ClerkProvider>
  );


  this code means if the user get signIn the go to the tap
